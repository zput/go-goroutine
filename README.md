# Introduction



（1）安装node版本管理模块n

sudo npm install n -g
下边步骤请根据自己需要选择

（2）安装稳定版

sudo n stable
（3）安装最新版

sudo n latest
（4） 版本降级/升级

sudo n 版本号//例如：sudo n 9.1.7
