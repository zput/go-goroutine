1.1. 大小端

  • 对于这两种存储方式，如果联系下我们常用 “异或” 操作，就能得到一个很好的记忆规则。总结来说就是：1
    • 低地址存储低字节即为小端存储；高地址存储高字节即为小端存储；
    • 低地址存储高字节即为大端存储；高地址存储低字节即为大端存储；

我们把 “低” 、“小” 认作 “0”， 把 “高”、“大” 认作 “1”，这样就可以利用 “异或” 的结果来加强记忆了。

内存 字节 大端/小端
低地址 低字节 小端存储
0 0 0
高地址 高字节 小端存储
1 1 0
低地址 高字节 大端存储
0 1 1
高地址 低字节 大端存储
1 0 1
